This is an H1 heading

The H1 heading is used for the main title of a webpage or document.

This is an H2 heading

An H2 heading is used for major sections or subsections that support the main topic.

This is an H3 heading

H3 headings are used for subsections within H2 sections.

This is an H4 heading

An H4 heading is used for smaller subsections within an H3 section.

This is an H5 heading

H5 headings are generally used for very specific details or small subsections within an H4 section.

This is an H6 heading

The H6 heading is the smallest and least commonly used.
